Web Accessibility Checklist
User Characteristic Accessibile design tip
Unable to see Code all images with ALT text
Unable to perceive colors Avoid using color alone to convey information
Unable to use mouse Be sure all website features can be accessed using keyboard
Unable to hear Add captions to multimedia
Prone to having seizures Avoid content that flashes
Easily distractible Keep the design simple

Accessibility Review of the Girard School District Home Page

Navigation and Orientation 72% Pass

The navigation and orientation of the site is horrible, both being my opinion, but also the truth showed by WAVE. The site doesn't have any formal organization, pages are just linked to one another and it would be hard for a person with dissabilities to navigate the site.

Text Equivalents 95% Pass

In most areas on the site, a text equivalent is shown. By this I mean that there is alt text shown for most links and images on the site. Also, there is alt-text on most navigation text.

Scripting 100% Pass

The site passed with 100% acceptance in the scripting testing area. But, in terms of the site, I don't think the site uses much javascript, and therefore I think that the results could have been skewed.

Styling 90% Pass

The overall styling of the site was decent, but there were some issues. Some of the issues were "simple" issues that included font and center selectors, bold styling issues, using 'i' instead of em, etc. A lot of these changes could be fixed with some simple knowledge.

HTML Standards 90% Pass

The only main issue with the site is that it didn't have the content type set and also, it didn't have the doctype for HTML 5.
