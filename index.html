Units of This Course

Unit 2 completed on Feburary 3, 2015

I would say that Unit 2 was a good refresher of my HTML coding lingo. I have learned a few new things throughout Unit 2, such as table scopes, and table captions - which I had no clue that they existed. Scopes are nice, since in the table using the table column headers can bold text, and if you only use table headers alone, it will not draw attention to your table columns.

I also learned in Unit 2 that Google Chrome does not support captions locally, instead you must actually upload your website to a public server and access it that way. It will then allow for you to view the captions. The below items in the list are really good for quick access to see common tags, and view quick descriptions.

  • Table Scopes can be used by rows and columns
  • Common HTML Tags
  • HTML
  • W3Schools

Unit 3 completed on Feburary 16, 2015

Unit 3 was a great refresher on CSS including Typography and page layout. I was able to very quickly go ahead and start creating the site that I wanted from previous experience. I was able to pick up from where I had previously left off on my own time.

One thing that I did learn from this lesson, is that typography can be selected. If the first font isn't there, it will then look for the next font that is in the list of fonts. This makes it so that the website can display on every possible device, allowing for a great website.

  • Pseudo Class Selectors
  • Good / Bad Typography
  • Site Layout
  • CSS Positioning
  • W3 Schools CSS Reference
  • Typography
