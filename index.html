Units of This Course

Unit 2

I would say that Unit 2 was a good refresher of my HTML coding lingo. I have learned a few new things throughout Unit 2, such as table scopes, and table captions - which I had no clue that they existed. Scopes are nice, since in the table using the table column headers can bold text, and if you only use table headers alone, it will not draw attention to your table columns.

I also learned in Unit 2 that Google Chrome does not support captions locally, instead you must actually upload your website to a public server and access it that way. It will then allow for you to view the captions. The below items in the list are really good for quick access to see common tags, and view quick descriptions.
